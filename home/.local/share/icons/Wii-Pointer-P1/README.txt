Created March 2024
Updated September 2025

Hai, thank you for downloading my cursor pack!
Feel free to use this set for whatever you want, but please seek my permission if you wish to redistribute it.

If you're looking for high quality SVG files of the Wii Pointers, they're also available at my site (https://primmr.dev)

These cursors should hopefully work in any environment that has Xcursor support.
The cursors now include scaling support, with support for the following pixel sizes: 96, 72, 48, 36, 30, 24.


How to install:
Firstly, unzip this archive if you haven't already

Choose one (or many) of the 4 colour schemes that you'd like to use,
then copy the corresponding Wii-Pointer-Px directory into ~/.icons/
(You may need to create the directory if it doesn't exist)

Then you need to install the theme - this will vary between distros and desktop environments,
and may require the use of external programs, so a search engine will be your friend!
(e.g. GNOME requires the installation of GNOME Tweaks)

I hope you enjoy the cursors!
